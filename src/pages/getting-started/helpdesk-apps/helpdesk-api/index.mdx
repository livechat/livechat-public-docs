# HelpDesk API

<a href="https://api.helpdesk.com/docs" target="_blank">HelpDesk API</a> is necessary when you build or improve your HelpDesk applications. That is why it's fully accessible to experiment with and discover its broad potential. The resources that can be managed using HelpDesk API are:

- <a href="https://api.helpdesk.com/docs#tag/Agents" target="_blank">Account and users</a>
- <a href="https://api.helpdesk.com/docs#tag/Tickets" target="_blank">Tickets</a>
- <a href="https://api.helpdesk.com/docs#tag/Email-domains" target="_blank">Managing your emails</a>
- <a href="https://api.helpdesk.com/docs#tag/Canned-responses" target="_blank">Automating your work</a>
- <a href="https://api.helpdesk.com/docs#tag/Reports" target="_blank">Analyzing your reports</a>

To read more about public API for HelpDesk, go to the <a href="https://api.helpdesk.com/docs" target="_blank">HelpDesk API Documentation</a>.
