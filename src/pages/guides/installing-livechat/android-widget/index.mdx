# Introduction

Add real-time customer support to your Android application with LiveChat's native SDK. This guide covers installation, setup, and advanced features.

<Warning>
ðŸ’¡ The SDK is now Kotlin-based and uses the new package `com.livechatinc.chatsdk`. See migration notes if you are upgrading from v2.x.x.
</Warning>

# Requirements

- Android 5.0 (API level 21) or higher
- Java 8 or higher

# Installation

## 1. Add the JitPack repository

Add JitPack to your root `build.gradle`:

<Tabs
  groupId="language"
  defaultValue="kotlin"
  values={[
    {label: 'Kotlin', value: 'kotlin'},
    {label: 'Java', value: 'java'},
  ]}>
<TabItem value="kotlin">

```kotlin
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
</TabItem>
<TabItem value="java">

```java
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
</TabItem>
</Tabs>

## 2. Add the dependency

Add to your app's `build.gradle`:

<Tabs
  groupId="language"
  defaultValue="kotlin"
  values={[
    {label: 'Kotlin', value: 'kotlin'},
    {label: 'Java', value: 'java'},
  ]}>
<TabItem value="kotlin">

```kotlin
dependencies {
    implementation 'com.github.livechat:chat-window-android:3.0.0'
}
```
</TabItem>
<TabItem value="java">

```java
dependencies {
    implementation 'com.github.livechat:chat-window-android:3.0.0'
}
```
</TabItem>
</Tabs>

## 3. Add Internet permission

Add to your `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.INTERNET" />
```

# Customer Information

Pre-fill the pre-chat form and provide customer details:

<Tabs
  groupId="language"
  defaultValue="kotlin"
  values={[
    {label: 'Kotlin', value: 'kotlin'},
    {label: 'Java', value: 'java'},
  ]}>
<TabItem value="kotlin">

```kotlin
LiveChat.getInstance().setCustomerInfo(
    "Joe", // Name
    "joe@mail.com", // Email
    "0", // Group ID, defaults to "0"
    Collections.singletonMap("internalCustomerId", "ABC123") // Custom params
)
```
</TabItem>
<TabItem value="java">

```java
Map<String, String> customParams = new HashMap<>();
customParams.put("internalCustomerId", "ABC123");
LiveChat.INSTANCE.getInstance().setCustomerInfo(
    "Joe", // Name
    "joe@mail.com", // Email
    "0", // Group ID
    customParams // Custom params
);
```
</TabItem>
</Tabs>

> **Note:** Call this before `LiveChat.show()`.

# UI Customization

## Localizing Text

Override string resources in your app's `strings.xml`:

```xml
<string name="failed_to_load_chat">Couldn\'t load chat.</string>
<string name="cant_share_files">File sharing is not configured for this app</string>
<string name="reload_chat">Reload</string>
```

## Custom Error View

Override `live_chat_error_layout.xml` in your layout resources.
Your custom view must contain a `View` with `live_chat_error_button` id.

## Customizing Activity

The chat activity follows your app's theme.
Override the activity declaration in your `AndroidManifest.xml` if needed.
