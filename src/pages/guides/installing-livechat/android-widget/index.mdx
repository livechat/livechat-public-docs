# Introduction

Add real-time customer support to your Android application with LiveChat's SDK. This guide covers installation, setup, and advanced features.

<Warning>

ðŸ’¡ The SDK is now Kotlin-based and uses the new package `com.livechatinc.chatsdk`. See migration notes if you are upgrading from v2.x.x.

</Warning>

# Requirements

- Android 5.0 (API level 21) or higher
- Java 8 or higher

# Installation

## 1. Add the JitPack repository

Add JitPack to your root `build.gradle`:

<Code>
<CodeSample path={'ADD JITPACK REPOSITORY'}>
```kotlin
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
</CodeSample>
</Code>

## 2. Add the dependency

Add to your app's `build.gradle`:

<Code>
<CodeSample path={'ADD DEPENDENCY'}>
```kotlin
dependencies {
    implementation 'com.github.livechat:chat-window-android:3.0.0'
}
```
</CodeSample>
</Code>

## 3. Add Internet permission

Add to your `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.INTERNET" />
```

# Usage

## Initialize the SDK

Initialize in your `Application`'s `onCreate()`:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
LiveChat.initialize("<LICENSE>", this)
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
LiveChat.initialize("<LICENSE>", this);
```
</CodeSample>
</Code>

## Show the Chat

Show the chat to a customer:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
LiveChat.getInstance().show()
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
LiveChat.getInstance().show();
```
</CodeSample>
</Code>

# Customer Information

Pre-fill the pre-chat form and provide customer details:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
LiveChat.getInstance().setCustomerInfo(
    "Joe", // Name
    "joe@mail.com", // Email
    "0", // Group ID, defaults to "0"
    Collections.singletonMap("internalCustomerId", "ABC123") // Custom params
)
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
Map<String, String> customParams = new HashMap<>();
customParams.put("internalCustomerId", "ABC123");
LiveChat.getInstance().setCustomerInfo(
    "Joe", // Name
    "joe@mail.com", // Email
    "0", // Group ID
    customParams // Custom params
);
```
</CodeSample>
</Code>

> **Note:** Call this before `LiveChat.show()`.

# Unread Message Counter

React to new messages using a listener:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
LiveChat.getInstance().newMessageListener =
    NewMessageListener { message, isChatShown ->
        // Handle new message
    }
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
LiveChat.getInstance().setNewMessageListener((chatMessage, isChatShown) -> {
    // Handle new message
});
```
</CodeSample>
</Code>

# UI Customization

## Localizing Text

Override string resources in your app's `strings.xml`:

```xml
<string name="failed_to_load_chat">Couldn\'t load chat.</string>
<string name="cant_share_files">File sharing is not configured for this app</string>
<string name="reload_chat">Reload</string>
```

## Custom Error View

Override `live_chat_error_layout.xml` in your layout resources.
Your custom view must contain a `View` with `live_chat_error_button` id.

## Customizing Activity

The chat activity follows your app's theme.
Override the activity declaration in your `AndroidManifest.xml` if needed.

# Handling Links

Intercept links sent in chat:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
LiveChat.getInstance().urlHandler =
        UrlHandler { url ->
            // Return true if handled, false to open in browser
            return@UrlHandler false
        }
    }
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
LiveChat.getInstance().setUrlHandler(uri -> {
    // Return true if handled, false to open in browser
    return false;
});
```
</CodeSample>
</Code>

# Advanced Usage

## LiveChatView Lifecycle Modes

Control the lifecycle of the chat view:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
LiveChat.initialize("<LICENSE>", this, LiveChatViewLifecycleMode.WHEN_SHOWN)
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
LiveChat.getInstance().initialize("<LICENSE>", this, LiveChatViewLifecycleMode.WHEN_SHOWN);
```
</CodeSample>
</Code>

## Embedding LiveChatView

Add to your layout XML:

```xml
<com.livechatinc.chatsdk.src.presentation.LiveChatView  />
```

Attach to activity or fragment:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
liveChatView.attachTo(this)
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
liveChatView.attachTo(this);
```
</CodeSample>
</Code>

Initialize with a listener:

<Code>
<CodeSample path={'Kotlin'}>
```kotlin
liveChatView.init(initCallbackListener)
```
</CodeSample>
</Code>
<Code>
<CodeSample path={'Java'}>
```java
liveChatView.init(initCallbackListener);
```
</CodeSample>
</Code>
