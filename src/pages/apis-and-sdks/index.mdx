# Introduction

This document summarizes all the APIs & SDKs helping you develop integrations built upon the Platform.

Many LiveChat and HelpDesk applications need our APIs or SDKs to be extended and become easy to use and efficient for their users. That is why our APIs and SDKs are fully accessible, and app developers can experiment with them.

# APIs

The list of APIs you can play with and develop your integrations:

- <a href="https://developers.livechat.com/docs/messaging/agent-chat-api" target="_blank">Agent Chat API</a> - interact as an agent in LiveChat App.
- <a href="https://developers.livechat.com/docs/messaging/customer-chat-api" target="_blank">Customer Chat API</a> - interact as a customer in LiveChat App.
- <a href="https://developers.livechat.com/docs/extending-chat-widget/javascript-api" target="_blank">Chat Widget JS API</a> - interact with the Chat Widget embedded on your site.
- <a href="https://developers.livechat.com/docs/management/configuration-api" target="_blank">Configuration API</a> - set up properties or webhooks.
- <a href="https://api.helpdesk.com/docs" target="_blank">HelpDesk API</a> - develop HelpDesk App.
- <a href="https://developers.livechat.com/docs/data-reporting/reports-api" target="_blank">Reports API</a> - interact with data.
- <a href="https://developers.livechat.com/docs/authorization/global-accounts-api" target="_blank">Global Accounts API</a> - manage options and accounts for multiple products.
- <a href="https://developers.livechat.com/docs/authorization/customer-accounts-api" target="_blank">Customer Accounts API</a> - authenticate and authorize processes of customers in LiveChat.
- <a href="https://developers.livechat.com/docs/monetization/billing-api" target="_blank">LiveChat Billing API</a> - charge for your paid apps.
- <a href="https://developers.livechat.com/docs/monetization/partner-program-api" target="_blank">Partner Program API</a> - build tools and make better use of the data.

# SDKs

The list of SDKs you can use to extend and customize your integrations:

- <a href="https://developers.livechat.com/docs/authorization/sign-in-with-livechat" target="_blank">Accounts SDK</a> - implement authorization.
- <a href="https://developers.livechat.com/docs/extending-agent-app/products-sdk/agent-app-sdk" target="_blank">Agent App SDK</a> - extend the LiveChat App.
- <a href="https://developers.livechat.com/docs/extending-agent-app/products-sdk/helpdesk-sdk" target="_blank">HelpDesk SDK</a> - extend HelpDesk App.
- <a href="https://developers.livechat.com/docs/extending-chat-widget/customer-sdk" target="_blank">Customer SDK</a> - build a custom widget.
- <a href="https://developers.livechat.com/docs/messaging/chat-sdk" target="_blank">Chat SDK</a> - build your apps quickly and in an easy way.
- <a href="https://developers.livechat.com/docs/messaging/go-sdk" target="_blank">LiveChat Go SDK</a> - build external backend apps.
- <a href="https://developers.livechat.com/docs/messaging/js-sdk" target="_blank">LiveChat JS SDK</a> - build external backend apps.
- <a href="https://developers.livechat.com/docs/messaging/python-sdk/#introduction" target="_blank">LiveChat Python SDK</a> - build external backend apps.
